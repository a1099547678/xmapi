### 手机产品接口

说明：

[]:为可选

- redmi：红米手机数据
- television：电视数据
- jotter：笔记本电脑
- appliances：家电
- wifi：路由器
- parts：智能硬件
- rimproduct：小米周边
- headset：音箱耳机
- livebox：生活箱包

请求URL：

​	http://localhost:3000/product/[xiaomi]

请求方式：

​	GET

参数：

| 参数名 | 必选 | 类型   | 说明             | 备注 |
| ------ | ---- | ------ | ---------------- | ---- |
| begin  | 是   | number | 从第几条数据开始 |      |
| ending | 是   | number | 截至到第几条数据 |      |
